域名本人使用了阿里云的域名。（9元）

主机方面原计划自搭服务器，但由于资金问题，不得不使用GitHub Pages来搭建网站。（0元）

GitHub Pages的坑：1.只支持静态网页. 2.国内访问速度缓慢

对于1我们只能乖乖地使用静态站了，而2很容易解决。我们需要一个CDN解决问题。

CDN我们选择的是CloudFlare。免费cdn+免费ssl证书，太香了！（0元）

就这样，我们的网站就建立成功了。共动用资金9元。还是很划算的。
